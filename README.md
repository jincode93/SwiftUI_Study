## JackPotApp(돌림판 만들기 실습 과정)
### 참여 인원 수를 정하고 버튼을 누르면 인원 수 안에서 랜덤한 번호를 결과로 보여주는 앱
#
## Design Ver.1 바뀐점
### 1. 이미지 개선(잭팟머신 이미지 추가 및 Picker부분 변화
### 2. 애니메이션이 동작중일 때 버튼을 누르면 앱이 꺼지는 오류가 발생하여 
###    애니메이션 동작중에는 버튼 비활성화 및 Picker 비활성화 기능 추가
#
